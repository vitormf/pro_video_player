# Pro Video Player Example

Example app demonstrating all features of the pro_video_player library.

## Running the Example

```bash
# From the repository root
cd example
flutter run
```

## Features Demonstrated

The example app showcases:

- **Video Sources** - Network, asset, and playlist sources
- **Playback Controls** - Play, pause, seek, volume, speed
- **Picture-in-Picture** - Enter/exit PiP mode
- **Background Playback** - Continue audio when app is backgrounded
- **Subtitles** - External and embedded subtitle tracks
- **Quality Selection** - HLS adaptive streaming quality control
- **Scaling Modes** - Fit, fill, and stretch options
- **Fullscreen** - Native fullscreen support
- **Playlists** - Multi-video playlist navigation
- **Multiple Players** - Simultaneous video playback
